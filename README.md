## About
Independent Google Ads consultant based in France.
This page describes an internal tool built using the Google Ads API.

## Tool purpose
The tool is used internally to manage and optimize multiple Google Ads accounts
under a Google Ads Manager Account (MCC).

Main functionalities:
- Automated reporting
- Account monitoring and alerting
- Controlled bulk actions (labels, budget pacing within predefined limits)

## Access
This tool is not a public product.
Access is restricted to internal users only (the consultant and, if applicable, NDA-bound contractors).

## Advertiser control
Advertisers retain full ownership of their Google Ads accounts.
They explicitly grant access via the Google Ads manager relationship
and can revoke access at any time.

Upon request, accounts are disassociated and all related API access is revoked
within 3 business days.
